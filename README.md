# ПорулиКа - Сайт квест-игры

Официальный сайт для юмористического квеста на ВАЗ-2107 "ПорулиКа".

## Описание

Сайт представляет информацию о квесте, позволяет записаться на участие и приобрести мерч. Дизайн выполнен в светлых, забавных тонах с элементами ностальгии по 90-м и автошколе.

## Функциональность

- Адаптивный дизайн
- Интерактивная навигация
- Форма записи на квест
- Секции с информацией об этапах квеста
- Галерея мерча
- Отзывы участников

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере

Или разместите на GitHub Pages.
